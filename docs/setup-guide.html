SMS Relay 설정 가이드

메르카리 SMS 인증번호 자동 전달 설정

0 Tailscale 설치 및 연결

Play Store에서 Tailscale 앱을 설치하세요.

설치 후 브라우저에서 아래 링크를 열어주세요:

브라우저에 입력
https://login.tailscale.com/auth/key?key=YOUR_AUTH_KEY

* 실제 auth key는 별도로 전달받은 것을 사용하세요

연결 완료 후 Tailscale 앱에 Connected 표시 확인

1 SMS Relay 앱 설치

아래 링크를 폰 브라우저에서 열어 APK를 다운로드하세요:

github.com/.../app-debug.apk

다운로드 완료 후→파일 열기→설치

"출처를 알 수 없는 앱" 경고가 뜨면 허용을 눌러주세요.
2 권한 허용

앱을 처음 열면 아래 권한을 요청합니다. 모두 허용해 주세요:

✓ SMS 수신 (RECEIVE_SMS)

✓ SMS 읽기 (READ_SMS)

✓ 전화 상태 (READ_PHONE_STATE) — SIM 구분용

✓ 알림 (POST_NOTIFICATIONS)

4 SIM 라벨 설정

듀얼 SIM 폰만 해당됩니다. SIM이 1개인 폰은 건너뛰세요.

각 SIM 슬롯에 해당하는 메르카리 계정명을 입력하세요:

SIM Labels (dual-SIM)
SIM 1:
saruwa2
Save
SIM 2:
johnleeknb
Save

* 계정명은 별도로 전달받은 것을 입력하세요

* 각각 입력 후 Save 버튼을 꼭 눌러주세요

5 메르카리 트리거 추가

가장 쉬운 방법: 다이얼로그 하단의 Mercari OTP 버튼을 누르면 자동으로 추가됩니다.

SIM이 1개인 폰은 Label에 계정명(예: saruwa2)을 직접 입력한 후 Add를 눌러주세요.
6 스위치 켜기

상단의 마스터 스위치를 ON으로 전환하세요:

SMS Relay

Listening — 2 rules active

상단 알림바에 "SMS Relay — Listening" 알림이 표시되면 정상 작동 중입니다.

앱 하단의 Recent Activity 섹션에서 전달 상태를 확인할 수 있습니다:

참고사항

• 기존 MacroDroid 설정은 그대로 유지하세요 (백업용)

• 앱이 종료되어도 SMS 수신은 계속 동작합니다

• 배터리 최적화에서 SMS Relay를 제외해 주세요

• Tailscale은 항상 연결 상태를 유지해야 합니다

• 문제 발생 시 앱 내 로그를 캡처해서 보내주세요
